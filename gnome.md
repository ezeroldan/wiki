# Gnome

## Gnome Tool
```BASH
sudo apt install -y gnome-tweak-tool
sudo apt install -y chrome-gnome-shell
mkdir .icons
mkdir .themes

sudo apt install gnome-session
sudo update-alternatives --config gdm3-theme.gresource
```
