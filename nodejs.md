# NodeJS

## Instalar
```BASH
sudo apt update -y && sudo apt upgrade -y
curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -
sudo apt install -y nodejs
sudo apt install -y npm
```

## Vue

### Instalar
```BASH
sudo npm install -g @vue/cli
```

### Actualizar
```BASH
sudo npm update -g @vue/cli
```
